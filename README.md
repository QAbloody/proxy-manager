# 🌐 Proxy Manager - Профессиональный менеджер прокси

![Python](https://img.shields.io/badge/Python-3.10%2B-blue)
![License](https://img.shields.io/badge/License-MIT-green)
![Status](https://img.shields.io/badge/Status-Active-brightgreen)

Полнофункциональное приложение для управления прокси-серверами с асинхронной проверкой, интеграцией API и кэшированием.

## ✨ Основные возможности

- ✅ **Загрузка прокси** из файлов и открытых API
- ✅ **Асинхронная проверка** с параллельными запросами (до 10 одновременно)
- ✅ **Измерение скорости** (ping/latency)
- ✅ **Автоматический выбор** лучшего прокси
- ✅ **Сортировка по скорости**
- ✅ **Сохранение результатов** в JSON
- ✅ **Кэширование** для быстрой загрузки
- ✅ **Полное логирование** в файл и консоль
- ✅ **Красивый CLI интерфейс** с цветным выводом
- ✅ **Обработка ошибок** и таймауты

## 📋 Требования

- Python 3.10+
- pip

## 🚀 Установка

### 1. Клонируем репозиторий

```bash

```

### 3. Установляем зависимости

```bash

```

## ⚙️ Конфигурация

Отредактируйте `config.py` для настройки:

```python
CHECK_TIMEOUT = 5           # Таймаут проверки (сек)
MAX_CONCURRENT = 10         # Макс одновременных запросов
TEST_URL = "http://httpbin.org/ip"  # URL для проверки
FREE_PROXY_API = "https://..."      # API источник прокси
```

## 📄 Формат файла прокси

Файл `proxies.txt` должен содержать прокси в формате `ip:port`, по одному на строку:

```
# Комментарии начинаются с #
1.10.188.149:8080
1.32.59.217:47045
103.153.232.182:8080
# Добавьте ваши прокси выше
```

## 🔍 Модули

### ProxyLoader
Загружает прокси из:
- Локальных файлов (`proxies.txt`)
- Открытого API (proxy-list.download)

### ProxyChecker
Проверяет прокси асинхронно:
- HTTP запросы к тестовому URL
- Измерение времени ответа
- Обработка таймаутов и ошибок
- Ограничение одновременных запросов

### ProxyManager
Управляет списком прокси:
- Сортировка по скорости
- Автоматический выбор лучшего
- Ручной выбор по индексу
- Статистика
- Сохранение/загрузка результатов

### Logger
Логирование в:
- Файл (`proxy_manager.log`)
- Консоль (реал-тайм)

## 🧪 Тестирование

Для тестирования используются бесплатные открытые прокси. **Внимание**: бесплатные прокси часто нестабильны и медленны.

Для продакшена рекомендуется использовать платные прокси-сервисы.
